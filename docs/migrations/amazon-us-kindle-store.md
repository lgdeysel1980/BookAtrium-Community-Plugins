# Amazon US Kindle Store

The Amazon US Kindle Store plugin moved from the BookAtrium community
catalogue to the official BookAtrium plugin catalogue.

Plugin ID:

`com.practicore.bookatrium.store.amazon-us-kindle`

Existing installations do not require reinstallation. BookAtrium matches the
installed plugin by its permanent plugin ID and checks future updates through
the official plugin registry.

Official catalogue:

https://github.com/lgdeysel1980/BookAtrium/tree/main/plugins/official/stores/amazon-us-kindle-store

Official release:

https://github.com/lgdeysel1980/BookAtrium/releases/tag/plugin-store-amazon-us-kindle-v1.0.4

The original standalone v1.0.4 release remains available temporarily during
the repository transition.
